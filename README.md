# CoreKit Sheet

A discardable action sheet.

## Usage

```
pod repo add corekit git@github.com:CoreKit/CocoaPods.git
```

## Pod cheatsheet

```
pod lib lint Sheet.podspec 
git tag 1.0.0
git push --tags
pod repo push corekit Sheet.podspec
```
