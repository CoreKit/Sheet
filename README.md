# CoreKit Sheet

A discardable action sheet.

## Usage

```
pod repo add corekit https://github.com/CoreKit/Sheet.git

source 'https://github.com/CoreKit/CocoaPods.git'
pod 'Sheet'

```

## Pod cheatsheet

```
pod lib lint Sheet.podspec 
git tag 1.0.0
git push --tags
pod repo push corekit Sheet.podspec
```
